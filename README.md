# Personal Portfolio Website

A modern and responsive personal portfolio website developed using HTML5, CSS3, and JavaScript to showcase technical skills, projects, achievements, and contact information.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Smooth scrolling navigation
- Animated skill progress bars
- Interactive project showcase section
- Contact form interface
- Modern UI with gradients and animations
- Professional portfolio layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Sections Included

- Home
- About
- Skills
- Projects
- Contact

## Skills Demonstrated

### Frontend Development
- Semantic HTML5 structure
- Responsive CSS layouts
- Flexbox and Grid
- Media Queries
- CSS animations and transitions

### JavaScript Functionality
- DOM Manipulation
- Event Handling
- Scroll Effects
- Interactive Animations
- Form Handling

## Project Purpose

This project was developed to demonstrate frontend web development skills and create a professional online portfolio for showcasing projects, technical expertise, and achievements.

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in any modern browser
